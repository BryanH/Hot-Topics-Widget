Displays a manually-generated list of categories as links.

== Description ==

This widget will display a manually-generated list of categories as links. Names that are not categories will still display, but not be links.

== Installation ==

1. Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation

2. Activate the Plugin from Plugins page.
